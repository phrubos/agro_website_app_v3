# AgroLab Logo Design Prompt

## Általános Koncepció
Modern, professzionális logo egy mezőgazdasági laboratóriumnak, amely precíziós gazdálkodásra és tudományos vizsgálatokra szakosodott.

## Részletes AI Prompt (Midjourney / DALL-E / Stable Diffusion)

```
Professional logo design for "AgroLab" agricultural laboratory.
Minimalist, modern design combining agricultural and scientific elements.

VISUAL ELEMENTS:
- Geometric leaf icon merged with laboratory beaker or microscope symbol
- Clean hexagonal or circular badge shape
- Optional: Subtle grid pattern or topographic lines suggesting precision agriculture
- Optional: DNA helix or molecular structure integrated into organic shapes

STYLE:
- Flat design, vector-style
- Professional and trustworthy
- Clean lines, balanced composition
- Modern tech-agriculture fusion aesthetic

TYPOGRAPHY:
- Company name: "AgroLab"
- Font style: Bold, geometric sans-serif (similar to Montserrat)
- Letter spacing: Slightly wide for modern feel
- Can be integrated with icon or placed below/beside

COLOR PALETTE (Primary):
- Deep Forest Green: #2D5016 (main color)
- Teal Accent: #1A936F (secondary/highlight)
- White: #FFFFFF (for contrast and text on dark backgrounds)

Alternative color combinations:
- Gradient option: Deep green (#2D5016) to teal (#1A936F)
- Monochrome version should work well in pure black/white

COMPOSITION OPTIONS:
1. Icon + Text horizontal lockup
2. Icon above text vertical lockup
3. Icon integrated into lettermark
4. Badge/emblem style with circular frame

MUST AVOID:
- Overly decorative or ornate details
- Cartoon or playful style
- Muddy or unclear symbolism
- Too many colors or gradients
- Old-fashioned agricultural clichés

TECHNICAL SPECS:
- Scalable from favicon (16x16px) to billboard
- Works in full color, monochrome, and white on dark
- Clear at small sizes
- Professional enough for laboratory reports
- Modern enough for tech/agriculture website

MOOD/FEELING:
Scientific, Precise, Trustworthy, Modern, Innovative, Natural, Professional, Clean
```

## Logo Variációk Amit Kérni Kell

1. **Teljes verzió** - Ikon + teljes szöveg
2. **Ikon csak** - Önmagában használható szimbólum (social media profilképhez)
3. **Szöveg csak** - Logotype verzió
4. **Vízszintes** - Széles formátumhoz
5. **Függőleges** - Magas formátumhoz
6. **Negatív** - Fehér verzió sötét háttérre
7. **Monokróm** - Fekete-fehér verzió

## Fájlformátumok

- **SVG** - Vektoros, végtelen skálázhatóság
- **PNG** - Átlátszó háttér (1000x1000px, 2000x2000px)
- **JPG** - Fehér háttér (különböző méretek)
- **ICO/Favicon** - 16x16, 32x32, 64x64px
